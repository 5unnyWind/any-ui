# ANY-UI

## 开发

# 请统一使用 pnpm：

请统一使用 pnpm：

```bash
# 建议全局安装 pnpm （或使用 npx
npm i -g pnpm
```

# 开发：

```bash
pnpm i
pnpm start
```

# 添加依赖：

```bash
# 生产依赖
pnpm add xxx
# 开发依赖
pnpm add -D xxx
```
