> 一套用于构建现代 Web 应用的开源设计敏捷方案

<br/>

## Quick Start

```bash
npm i any-ui
# or
yarn add any-ui
# or
pnpm add any-ui
```

```js
import { Button } from "any-ui";

export default () => (
  <>
    <h2>按钮</h2>
    <Button type="default">默认</Button>
  </>
);
```
